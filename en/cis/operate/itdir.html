目录管理

# 互感器目录维护

# 功能说明

对互感器生产厂家可供应的互感器清单进行新增、修改、删除。互感器目录维护涉及互感器的厂家、电压、电流、类型。

# 操作说明

# 互感器目录新增

（1）依次点击:&quot;资产管理》目录管理》互感器目录维护&quot;菜单。进入互感器目录维护界面,如图1.1.1互感器目录管理界面。

图1.1.1

（2）在互感器目录维护页面，点击界面【新增】按钮，进入到&quot;互感器参数新增界面&quot;，如图1.1.2。

图1.1.2

（3） 在互感器参数新增界面，填写生产厂家、类别、型号、额定电压、额定频率、电流变比、电压变比、电流互感器精度等级、电压互感器精度等级等互感器参数信息，点击【保存】按钮,添加互感器目录成功，在&quot;互感器目录信息&quot;界面可以查看到新增的互感器目录信息。

# 互感器目录修改

在&quot;互感器目录信息&quot;页面，选择一条需要编辑的互感器目录，点击【编辑】按钮，弹出编辑窗口，如图1.1.3，可修改生产厂家、类别、型号、额定电压、额定频率、电流变比、电压变比、电流互感器精度等级、电压互感器精度等级等互感器参数信息，点击【保存】按钮，修改成功后，返回&quot;互感器目录信息&quot;页面可以查看已变更的互感器目录信息。

    图1.1.3

# 互感器目录删除

在&quot;互感器目录信息&quot;界面选中一条活多条需要删除的互感器，点击【删除】按钮，系统弹出消息对话框并提示&quot;是否删除&quot;,点击【确认】按钮,系统提示删除成功，点击【取消】按钮,关闭窗口，不予删除。如图1.1.4消息对话框,所示:

图1.1.4

如图1.1.4提示框中，点击【确认】按钮，系统删除成功，如图1.1.5。

图1.1.5

# 互感器目录查询

在查询条件框中输入生产厂家、类别、型号等单个或多个查询条件，点击【查询】按钮，在互感器目录信息 展现符合查询条件的结果集信息。

# 互感器目录查询条件清空

在上方查询栏中输入生产厂商、类别、型号等单个或多个查询条件，点击【清空】按钮，可以清除输入的查询信息。

# 参考操作说明

【查询】：根据输入的查询条件，查询出符合条件的互感器目录。

【清空】：清除输入的查询条件。

【新增】：添加互感器目录信息。

【删除】：删除未使用的互感器目录。

【编辑】：修改未使用的互感器目录。

# 注意事项

1、互感器型号不可重复。

2、已被资产档案使用的互感器目录不允许进行修改，也不允许删除。

3、标注星号（*）的为必填项。
